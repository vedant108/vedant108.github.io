Made: 01/20/21

Object detection in Point Cloud is popular in HD Map and sensor-based autonomous driving. There are basically four types of object you can obtain in daily scenario: Road surface - painted lane marking and pavement area Support facility - road boundary, road sign, light pole, etc. Uncorrelated object - sidewalk, building, etc. Moving objects - pedestrian, vehicle, bicycle, etc. The goal of this project is to detect light poles.
